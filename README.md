# 🏕️ Campsite Commander

**Campsite Commander** is an Android app built with **Kotlin** and **Jetpack Compose** to help campers plan, organize, and manage their outdoor adventures.  
It simplifies campsite setup, gear tracking, and trip scheduling through a clean, intuitive interface.

---

## 🌟 Features
- Manage camping gear lists and supplies  
- Plan trips and track campsite details  
- Simple navigation between screens  
- Lightweight design optimized for mobile use  
- Built with modern Android architecture

---

## 🧩 Tech Stack
- **Language:** Kotlin  
- **Framework:** Jetpack Compose  
- **IDE:** Android Studio  
- **Version Control:** Git & GitHub
